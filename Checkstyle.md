## Checkstyle

This project uses a basic set of Checkstyle restrictions to enforce code style and readability.  
The rules are defined in the [checkstyle.xml](checkstyle.xml) file.  

The Checkstyle configuration includes the following restrictions:

- **Maximum line length:** 120 characters  

- **Indentation:** 4 spaces per level  

- **Method names:** lowerCamelCase  

- **Unused imports:** disallowed  

- **Maximum nested `for` loops:** 4 levels  

- **Maximum nested `if` statements:** 4 levels
